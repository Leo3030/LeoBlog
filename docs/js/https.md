## HTTP 的基本概念

HTTP：是互联网上应用最为广泛的一种网络协议，是一个客户端和服务器端`请求和应答的标准（TCP）`，用于WWWW服务器传输超文本到本地浏览器的`超文本传输协议`。

## HTTP工作原理
HTTP协议定义Web客户端如何从Web服务器请求Web页面，以及服务器如何把Web页面传送给客户端。客户端想服务器发送一个请求报文，服务器已一个状态行作为响应。

## HTTP请求/响应的步骤
- 客户连接到Web服务器
- 发送HTTP请求
- 服务器接收请求并返回HTTP响应
- 释放TCP连接
- 客户端（浏览器）解析HTML内容

## HTTP的5种方法
- GET---获取数据
- POST---传输资源
- PUT---更新资源
- DLELETE---删除资源
- HEAD---获取报文头部
